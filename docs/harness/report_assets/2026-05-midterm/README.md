# 2026-05 Midterm Report Assets

이 디렉터리는 중간보고서/발표자료에 넣을 curated screenshot과 figure를 모으는 위치다.

원칙:

- 원시 benchmark artifact를 그대로 연결하지 않는다.
- 캡처 후 token, cookie, API key, 계정명, 개인 브라우저 정보가 보이면 crop/redaction한다.
- Grafana 캡처는 대응되는 `pack_id`, push 시각, `summary.json`/`results.json`과 함께 기록한다.
- CAPTCHA 원본 이미지는 넣지 않는다. 차단 사례는 `BLOCKED_USER_ACTION` 같은 분류표로 설명한다.

권장 파일명은 `../REPORT_FIGURE_SOURCES_2026-05-10.md`를 따른다.
